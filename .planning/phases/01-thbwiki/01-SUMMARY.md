## One-Liner

THBWiki OpenSearch API integration with Caffeine cache layer, providing album search functionality via admin REST endpoint.

## Decisions Made

1. **Search API Choice**: Used MediaWiki OpenSearch API (`/api.php?action=opensearch`) instead of page scraping for Phase 1
2. **Service Location**: Placed ThbwikiService and ThbwikiCacheService in tongrenlu-dao module for reuse across modules
3. **Cache Strategy**: Caffeine with 24h TTL and 1000 max size for album caching
4. **Model Design**: Simple POJOs with Lombok @Data and Jackson @JsonInclude for JSON serialization
