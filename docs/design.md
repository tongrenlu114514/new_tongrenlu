# 东方Project同人音乐网站设计方案

## 设计理念

### 核心概念
- **东方美学与现代融合**: 将传统东方元素（樱花、竹子、水墨）与现代网页设计相结合
- **音乐可视化**: 通过视觉元素表达音乐的节奏和旋律感
- **神秘与优雅**: 营造东方Project特有的神秘氛围，同时保持界面的优雅简洁

### 颜色方案

  辅助色: #df8888 - 柔和的粉红色，用于导航链接悬停效果和主要按钮
  强调色: #00a6eb - 蓝色，用于特殊元素和悬停效果
- **文字色**: 
  - 主要文字: 白色 (#FFFFFF)
  - 次要文字: 浅灰蓝 (#D0D0E0) - 带微蓝调，与背景更协调
  - 链接文字: 樱花粉 (#FFA6B7)

### 排版系统
- **标题字体**: "Noto Serif SC" (衬线) - 用于主标题，体现东方韵味
- **正文字体**: "Noto Sans SC" (无衬线) - 用于正文内容，保证可读性
- **装饰字体**: "Noto Serif SC" (衬线) - 用于特殊标题和装饰文字
- **字体层级**:
  - H1: 48px, 粗体
  - H2: 36px, 中等
  - H3: 24px, 中等
  - 正文: 16px, 常规
  - 小字: 14px, 常规

## 视觉效果

### 背景效果
- **主背景**: 使用shader-park创建动态星空粒子效果
- **装饰元素**: 
  - 浮动的樱花花瓣 (使用Matter.js物理引擎)
  - 微妙的光晕效果
  - 渐变色彩流动

### 交互动画
- **页面加载**: 元素从下方淡入，带有轻微的Y轴位移
- **悬停效果**: 
  - 卡片: 3D倾斜 + 阴影扩展 + 轻微放大
  - 按钮: 颜色渐变 + 发光边缘
  - 图片: Ken Burns效果 (缩放 + 位移)
- **滚动动画**: 元素在进入视窗时触发淡入动画

### 音乐可视化
- **频谱分析**: 使用p5.js创建实时音频可视化
- **波形显示**: 动态波形图显示在播放器背景
- **粒子响应**: 背景粒子根据音乐节奏跳动

### 图标系统
- **音乐相关**: 使用自定义SVG图标
- **UI元素**: 采用线性图标风格
- **装饰图标**: 东方主题的装饰性图标（樱花、月亮、星星）

## 布局设计

### 网格系统
- **桌面端**: 12列网格系统
- **平板端**: 8列网格系统  
- **移动端**: 4列网格系统

### 间距规范
- **基础间距单位**: 8px
- **组件间距**: 16px, 24px, 32px
- **页面边距**: 桌面端48px，移动端16px

### 组件样式
- **卡片设计**: 
  - 圆角: 12px
  - 阴影: 0 4px 20px rgba(0,0,0,0.3)
  - 背景: 半透明白色 (rgba(255,255,255,0.1))
- **按钮设计**:
  - 主要按钮: 渐变背景 + 发光效果
  - 次要按钮: 透明背景 + 边框
  - 圆角: 8px

## 响应式设计

### 断点设置
- **移动端**: < 768px
- **平板端**: 768px - 1024px  
- **桌面端**: > 1024px

### 适配策略
- **音乐播放器**: 移动端采用全宽底部固定设计
- **网格布局**: 移动端单列，平板双列，桌面三列
- **导航菜单**: 移动端使用汉堡菜单

## 品牌元素

### Logo设计
- 结合"东"字与音符元素
- 使用樱花粉作为主色
