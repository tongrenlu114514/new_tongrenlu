# 东方Project同人音乐网站交互设计

## 核心交互功能

### 1. 音乐播放器系统
- **功能描述**: 创建一个完整的音乐播放器，支持播放、暂停、上一首、下一首、音量控制、进度条拖拽
- **交互细节**: 
  - 点击任意歌曲卡片即可开始播放
  - 播放器固定在页面底部，显示当前播放歌曲信息
  - 支持播放列表管理，用户可以添加/删除歌曲
  - 播放模式切换：顺序播放、随机播放、单曲循环
- **视觉效果**: 播放时显示音频可视化效果，使用p5.js创建动态音频频谱

### 2. 音乐发现与筛选系统
- **功能描述**: 多维度音乐筛选和搜索功能
- **交互细节**:
  - 左侧筛选面板：按社团、风格、角色、游戏作品分类
  - 中央网格展示：专辑封面卡片，支持悬停预览
  - 搜索框支持实时搜索，输入时显示建议结果
  - 点击筛选标签可以组合多个条件
- **数据模拟**: 预设20+个知名社团，15+种音乐风格，30+个角色分类

### 3. 个性化收藏系统
- **功能描述**: 用户可以创建个人收藏夹，保存喜欢的音乐
- **交互细节**:
  - 点击心形图标添加/移除收藏
  - 支持创建多个收藏夹（如"喜欢的摇滚曲"、"治愈系音乐"）
  - 收藏页面展示个人统计信息
- **本地存储**: 使用localStorage保存用户收藏数据

### 4. 艺术家档案系统
- **功能描述**: 展示各个同人社团和艺术家的详细信息
- **交互细节**:
  - 艺术家卡片网格展示
  - 点击艺术家查看详细页面：简介、代表作品、专辑列表
  - 支持按活跃时间、作品数量排序
- **内容**: 包含15+个知名社团的详细信息

## 页面结构规划

### 主页 (index.html)
- Hero区域：东方风格的视觉展示
- 特色音乐轮播
- 热门社团展示
- 最新上传音乐

### 音乐库 (music.html)
- 左侧：筛选面板
- 中央：音乐网格展示
- 右侧：播放队列
- 底部：音乐播放器

### 艺术家 (artists.html)
- 艺术家网格展示
- 详情页面模态框

### 关于页面 (about.html)
- 东方Project介绍
- 同人音乐文化
- 网站介绍

## 技术实现

### 使用的库和效果
- **Anime.js**: 页面元素动画、卡片悬停效果
- **p5.js**: 音频可视化、背景粒子效果
- **ECharts.js**: 音乐统计数据可视化
- **Splide.js**: 音乐专辑轮播
- **Matter.js**: 浮动物理效果（装饰用）

### 响应式设计
- 移动端优化的音乐播放器
- 触摸友好的交互元素
- 自适应网格布局

### 数据管理
- 模拟音乐数据库（JSON格式）
- 本地存储用户偏好
- 动态加载和筛选
